Excel circuit format

Each row describes one node and its primary connection. The first row must be the header row shown below. Upload the workbook to the viewer or POST it to the circuit API.

Required columns

Node Connects_To Type Ports Registers
UART APB_BUS Component clk,rst,tx,rx CTRL,STATUS
APB_BUS CPU Bus paddr,pwrite
CPU UART Component irq

Column reference

Column Description Required
Node Unique name of the component or bus instance. Yes
Connects_To Name of the node this row connects to (creates a directed edge). Yes
Type Component, Bus, CPU, Memory, etc. Yes
Ports Comma-separated port names (no spaces), e.g. clk,rst,tx,rx. Yes
Registers Comma-separated register names. Leave empty if none. No
Ports and registers use comma separation only — do not add spaces after commas. Type values are mapped to viewer categories (e.g. Bus → bus, Component → peripheral).
