ip-xact viewer

Import documentation

This viewer renders SoC block diagrams from architecture data. You can import Excel, XML (via API), or JSON directly.

Supported import methods

  • Excel (.xlsx / .xls) — uploaded to the circuit API and converted to JSON.
  • XML (.xml) — IP-XACT or circuit XML uploaded to the same API.
  • JSON — pasted or dropped; parsed locally without API call.

Circuit API

Excel and XML files are POSTed as multipart/form-data with field name file: 

POST https://python.verifplay.com/api/circuit

The API returns JSON with components and connections, which the viewer normalizes and lays out.

JSON shape (after conversion)

{
  "components": {
    "UART": {
      "type": "Component",
      "ports": ["clk", "rst", "tx", "rx"],
      "registers": ["CTRL", "STATUS"]
    }
  },
  "connections": [
    { "source": "UART", "target": "APB_BUS" }
  ]
}

Layout

Components are assigned to semantic layers (CPU → bus → memory → peripherals) and placed left-to-right. Best for roughly 10–150 component SoC diagrams.
